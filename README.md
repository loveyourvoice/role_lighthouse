Lighthouse
=========

Эта роль устанавливает лайтхаус на Centos

Переменные
--------------

| vars | description      |
|------|------------------|
| 0.1  | Версия лайтхауса |

Зависимости
------------

- GIT

Пример плейбука
----------------

Пример плейбука для использование данной роли:

    - hosts: servers
      roles:
         - { role: lighthouse }
